Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM.The easiest way to find an HTML element in the DOM is by using the element id. They are:

  • getElementById
  • getElementByClassName
  • getElementByTagName
  • querySelector
  • querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to sets HTML content inside an element.It returns all text, including html tags that is contained by an element.
innerText is used to sets the visible text of an element. It returns all text inside an element.
textContent is used to sets all text content, including hidden elements. It returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a parent element handles events for its child elements instead of attaching event listeners to each child individually.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a technique where an event starts at the target element and then propagates upwards through its ancestor elements, all the way to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create a new element by using document.createElement()
We can add element once it is created by using methods like:

  • appendChild()
  • append()
  • prepend()
  • innerHTML

We can remove elements by using methods like:

  • remove()
  • removeChild()
